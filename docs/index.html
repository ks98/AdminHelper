Projektumfang

simple remotemanager besteht aus mehreren lauffähigen Teilen, die getrennt deployt werden können, aber zusammen eine Team-Remote-Plattform bilden.

Desktop-Client

Tauri v2 App (Rust + HTML/CSS/JS) für SSH, RDP und Web-Verbindungen, inklusive lokaler Datenspeicherung, Sync und Ansible-Integration.

Server

FastAPI-Backend mit Web-UI, JWT-Login, Rollenmodell, API-Keys, Hooks, FRP-Tunnel, Ansible-Playbooks und TLS via Docker.

Monitoring + Agent

Separater Monitoring-Service mit VictoriaMetrics und Unified Go Agent für Ressourcen-Überwachung (CPU, RAM, Disk, Services, Docker, ZFS, Proxmox).

Chrome Extension

Browser-Popup für kind: web-Einträge, mit Cache, Suche, Tag-Gruppierung und API-Key-gestütztem Zugriff.

Architektur auf einen Blick

Ebene Technik Verantwortung
Desktop Rust, Tauri v2, Vanilla JS Verbindungsverwaltung, Start nativer Clients, lokaler Passwortspeicher (RDP)
Server API Python, FastAPI, SQLAlchemy, SQLite Auth, Nutzerverwaltung, API-Keys, Connection-CRUD, Hooks, FRP-Tunnel, Ansible-Playbooks
Server UI Statische SPA in server/frontend Admin-Oberfläche für Verbindungen, Benutzer, API-Keys, Hooks, Monitoring und Ansible
Monitoring-Service Python, FastAPI, APScheduler, VictoriaMetrics Check-Engine (Ping/TCP/HTTP/SNMP/Agent), Templates, Alerting (Webhook/E-Mail)
Agent Go, gopsutil, Cobra CLI Unified Agent für Linux + Windows: FRP-Config-Sync und Ressourcen-Monitoring
Browser Extension Manifest V3, Service Worker Anzeige/Öffnen von Web-Zielen aus Team-Server mit Hintergrund-Refresh

Schnellstart je Komponente

Server per Docker starten

cp .env.example .env
# SECRET_KEY und ADMIN_PASSWORD setzen

docker compose pull
docker compose up -d

Danach erreichbar unter https://localhost (Port-Mapping: 443 -> 8443).

Desktop-Client lokal starten

cd desktop/src-tauri
cargo tauri dev

Produktion-Build über cargo tauri build.

Server API-Dokumentation

Swagger-UI ist unter /api/docs verfügbar, z. B. https://localhost/api/docs.

Extension testen

In Chrome chrome://extensions öffnen, Entwicklermodus aktivieren, Entpackt laden und Ordner extension/ wählen.
