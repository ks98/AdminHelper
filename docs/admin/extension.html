Browser-Extension

Die Chrome/Edge-Extension (Manifest V3) zeigt deine AdminHelper-Verbindungen im Browser-Popup und öffnet Web-Verbindungen direkt in neuen Tabs. Ideal, wenn du häufig zwischen iDRAC, Proxmox-UIs und anderen Web-Interfaces springst.

Installation

  1. Extension-ZIP aus dem Web-Panel laden: Einstellungen → Extension herunterladen
  2. In Chrome/Edge: chrome://extensions öffnen, Entwicklermodus aktivieren, Entpackt laden und den Ordner wählen
  3. Alternativ direkt aus dem Chrome Web Store (sobald veröffentlicht)

Konfiguration

  1. Extension-Icon klicken → Einstellungen
  2. Server-URL eintragen (https://srm.example.com)
  3. API-Key aus dem Web-Panel einfügen (Scope: connections:read)
Refresh-Intervall

Die Extension pullt alle 5 Minuten einen frischen Snapshot der Verbindungsliste via Chrome-Alarm-API. Das ist bewusst konservativ gewählt, um Browser-Ressourcen zu sparen; manueller Refresh per Knopfdruck im Popup.

Was wird gezeigt?

Nur Verbindungen mit Kind http oder https. Andere Typen (RDP, SSH, VNC) werden ausgeblendet, weil sie im Browser ohnehin nicht direkt genutzt werden können – dafür ist der Desktop-Client zuständig.

Gruppen & Filter

Das Popup übernimmt die Gruppen-Struktur vom Server und erlaubt einfache Suche. Favoriten (Sternchen in der Web-UI) erscheinen als separater Abschnitt oben.

Updates

Die Extension prüft beim Start gegen die Server-API, ob eine neuere Version verfügbar ist. Wird das Update nicht automatisch ausgerollt (z. B. bei Unpacked-Installation), meldet die Extension es im Popup mit einem Download-Link.
