Admin Guide

Overview

This guide is aimed at operators who install, configure, and maintain AdminHelper. It covers the server, reverse proxy, agents, monitoring stack, FRP tunnel, and day-to-day operations.

Stack at a Glance

Server

FastAPI monolith with PostgreSQL 17 + Redis, 9 routers, Alembic migrations, APScheduler, hook system.

Web Panel

Svelte 5 SPA served by the server, hash router, i18n DE/EN.

Desktop Client

Tauri 2 + Rust + Svelte 5, OS keyring, RDP/SSH/VNC launcher.

Agents

Go-Agent (apps/agent/) for Linux & Windows – monitoring, FRP, provisioning.

Monitoring

Dedicated FastAPI service, VictoriaMetrics storage, agent push + scheduler pull.

FRP

Managed mTLS PKI (10 y CA / 1 y client certs), TCP/HTTP/STCP tunnels.

Reading Order

  1. Installation – Docker Compose setup, reverse proxy, first login.
  2. Operations & Config – Environment variables, data directories, backups, logging.
  3. Users & Access – Roles, API keys, rate limits, audit log.
  4. Agent Deployment – Install Linux/Windows agents, provision flow.
  5. Monitoring – Check types, alerts, charts.
  6. FRP Tunnel – PKI, visitor ports, certificate rotation.
  7. – Deployment for Chromium.
  8. Troubleshooting – Structured error diagnosis.
Prerequisites

Linux host, Docker & Docker Compose v2, a public port for reverse-proxy TLS termination, and optionally a wildcard DNS entry for FRP HTTP tunnels.
