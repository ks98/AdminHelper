Project Scope

Simple Remote Manager consists of several deployable parts.

Desktop Client

Tauri v2 app (Rust + HTML/CSS/JS) to manage SSH, RDP, and Web targets with Ansible integration.

Server

FastAPI backend with web UI, JWT auth, API keys, hooks, FRP tunnels, Ansible playbooks, and TLS runtime.

Monitoring + Agent

Separate monitoring service with VictoriaMetrics and unified Go agent for resource monitoring (CPU, RAM, Disk, Services, Docker, ZFS, Proxmox).

Chrome Extension

Fast access to kind: web connections from the shared server dataset.

Architecture at a Glance

Layer Technology Responsibility
Desktop Rust, Tauri v2, Vanilla JS Connection management, opening native clients, local credential handling.
Server API Python, FastAPI, SQLAlchemy, SQLite Authentication, users, API keys, connection CRUD, hooks, FRP tunnels, Ansible playbooks.
Server UI Static SPA in server/frontend Admin pages for connections, users, API keys, hooks, monitoring, and Ansible.
Monitoring Service Python, FastAPI, APScheduler, VictoriaMetrics Check engine (Ping/TCP/HTTP/SNMP/Agent), templates, alerting (webhook/email).
Agent Go, gopsutil, Cobra CLI Unified agent for Linux + Windows: FRP config sync and resource monitoring.
Browser Extension Manifest V3 Cached browsing/opening of shared web connections.

Quickstart

Start Server with Docker

cp .env.example .env
# set SECRET_KEY and ADMIN_PASSWORD

docker compose pull
docker compose up -d

Run Desktop Client

cd desktop/src-tauri
cargo tauri dev

Build with cargo tauri build.
